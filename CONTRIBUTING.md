# Contributing to GDGC RCCIIT Website

Thank you for your interest in contributing to the GDGC RCCIIT official website!

## Getting Started

1. Fork the repository
2. Clone your fork
3. Create a new branch for your feature
4. Make your changes
5. Submit a pull request

## Code Style

- Follow the existing code style
- Use TypeScript for type safety
- Write meaningful commit messages

## Questions?

Feel free to reach out to the GDGC RCCIIT team!
